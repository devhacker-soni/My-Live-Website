# Web Developer 🌐

A high-performance, multi-page website built from scratch. This project showcases my journey through the first 13 videos of Harry Bhai's Sigma Web Development Course, focusing on clean code, performance optimization, and conceptual learning.

## Pages
- **Home** — First website covering Frontend, Backend & Full Stack Developer concepts
- **About** — My journey, skills, and how I used Harry Bhai's videos + Claude AI to build this
- **Contact** — A fully functional form with name, email, phone, state, city & interest field

## Features And Learning
- Dark / Light Mode Toggle
- Back to Top Button
- Form Success Message
- SEO Optimized — 100% Lighthouse Score
- Core Web Vitals Optimized — CLS: 0, Performance: 98%
- Embedded YouTube video & CodeWithHarry website
- Code snippets with Prism.js syntax highlighting
- Responsive Navbar across all pages
